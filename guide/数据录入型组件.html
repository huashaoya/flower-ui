# switch组件

两种相互对立的状态的切换

# 基础用法

绑定一个 v-model 为Boolean类型的变量，可以得到切换的状态

# 自定义颜色

使用 activeColor、inActiveColor 属性来设置开与关的背景色

# 添加文字描述

未激活
已激活

使用 activeText、inActiveText 属性来设置开与关的文字描述

# Attributes

参数说明类型可选值默认值
v-model绑定值Boolean————
activeColorswitch打开时的背景色String——#00bc12
inActiveColorswitch关闭时的背景色String——#f0f0f4
activeTextswitch打开时的文字描述String————
inActiveTextswitch关闭时的文字描述String————

# upload组件

通过点击或者拖拽上传文件

# 按钮点击上传

拖拽或点击上传文件松开即可

btnShow属性设置是否显示按钮 change事件接受返回参数，即上传文件的相关信息

# 接受多值上传

拖拽或点击上传文件松开即可

multiple 属性可接受多个值的文件上传字段：

# 设置按钮样式

拖拽或点击上传文件松开即可

参考 fl-button 设置相应按钮参数

# 拖拽上传

拖拽或点击上传文件松开即可

设置dragShow属性设置是否显示拖拽区域

# Radio单选

更易于使用的单选组件

# 基础用法

使用v-model进行双向绑定，label指定标签

<fl-radio-group v-model="radioGroup">
  <fl-radio label="0">1</fl-radio>
  <fl-radio label="1">2</fl-radio>
  <fl-radio label="2">3</fl-radio>
</fl-radio-group>

# Attributes

参数说明类型可选值默认值
v-model绑定值String——''
label标签值String——''

# CheckBox多选

更易于使用的多选组件

# 基础用法

使用v-model进行双向绑定，label指定标签

<fl-checkbox-group v-model="checkboxGroup">
  <fl-checkbox label="html"></fl-checkbox>
  <fl-checkbox label="css"></fl-checkbox>
  <fl-checkbox label="javaScript"></fl-checkbox>
</fl-checkbox-group>

# Attributes

参数说明类型可选值默认值
v-model绑定值Array——[]
label标签值String——''
